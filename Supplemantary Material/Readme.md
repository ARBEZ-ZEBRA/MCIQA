This folder is the code related with MCIQA-2K: A Multi-Dimensional Dataset and No-Reference Quality Assessment Benchmark for Colorized Images.


MCIQA_CS_training.py and MCIQA_CS_test.py contain the code related with color smearing (CS) branch in MCIQA.

MCIQA_SCM_training.py and MCIQA_SCM_test.py contain the code related with semantic color misalignment (SCM) branch in MCIQA.

MCIQA_GN_training.py and MCIQA_GN_test.py contain the code related with global naturalness (GN) branch in MCIQA.

MCIQA_2K_CS_MOS_train.json and MCIQA_2K_CS_MOS_test.json contain the Z-score of CS dimension.

MCIQA_2K_SCM_MOS_train.json and MCIQA_2K_SCM_MOS_test.json contain the Z-score of SCM dimension.

MCIQA_2K_GN_MOS_train.json and MCIQA_2K_GN_MOS_test.json contain the Z-score of GN dimension.
